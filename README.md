# lakansyèl for Stylus

> Color Scheme utilities for stylus

**lakansyèl for Stylus** is a collection of utilities to generate color schemes in your stylesheets.

## Usage

Include **lakansyèl** in your stylus stylesheets with

```css
@import "lakansyel"
```
